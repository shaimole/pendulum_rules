# MagicValues

## Caster values

### Manipulation (MP)

(Intelligence + Perception) / 2

### Projection (PJ)

(Dexterity + Perception) / 2

### Protection (PT)

(Willpower + Perception) / 2

## Damage magic

### Magic (one round)

Spell damage + Intelligence

### Magic (two-round)

Spell Damage + (Intelligence x2)

### Powers (one-round)

Power damage + (Willpower / 2)

### Powers (two-round)

Power damage + willpower
